Security considerations

Encryption is important because it allows you to securely protect data that you don't want anyone else to have access to. Businesses use it to protect corporate secrets, governments use it to secure classified information, and many individuals use it to protect personal information to guard against things like identity theft. When you're communicating with someone else over the Internet, encryption is also your first line of defense. Authenticating on the SMTP server that handles your outgoing mail should only be done via an encrypted connection (and no SMTP server should operate without some attempt to authenticate users), so that a malicious security cracker "listening in" will not be able to pluck your username and password out of plain text communication.

Watching out for SQL injections is considerable as well. By using parameterised queries it can be easily prevented.
